# Example Expression Data for NMFprofileR

Example GBM Mesenchymal-like subtype expression data from
PanCancerAtlas, used to demonstrate the NMFprofileR workflow.

## Usage

``` r
example_expression_data
```

## Format

A data frame of non-negative expression values with 20501 rows (genes,
with gene symbols as rownames) and 53 columns (TCGA samples, with TCGA
barcodes as column names).

## Source

PanCancerAtlas (The Cancer Genome Atlas).
